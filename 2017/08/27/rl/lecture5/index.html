Q-Learning

off-policy learning of action-values Q(s,a)
No importance sampling is required
Next action is chosen using behaviour policy $A_{t+1} \sim
\mu(\cdot|S_t)$
But the estimation of next Q value using the target policy.
$$
Q(S_t,A_t) \rightarrow Q(S_t,A_t)+\alpha (R_{s,a} + \gamma Q(S_{t+1},A_{t+1})-Q(S_t,A_t))
$$

feature

  3. simplest possible idea: value = mean return
  4. all episodes must terminate

Temporal-Difference Learning

Feature

  1. no knownledge of MDP transitions
  2. learns from incomplete episodes: bootstrapping
  3. simplest possible idea: value = mean return
  4. TD updates a guess towards a guess

    update $V(S_t)$ for TD(0)

    $$
    V(S_t) \leftarrow V(S_t)+\alpha (R_{t+1}+\gamma V(S_{t+1})-V(S_t))
    $$

TD($\lambda$)

Replay Memory

We’ll be using experience replay memory for training our DQN. It stores the transitions that the agent observes, allowing us to reuse this data later. By sampling from it randomly, the transitions that build up a batch are decorrelated. It has been shown that this greatly stabilizes and improves the DQN training procedure.
