BrainMatrix README

2016.11.11 LiuXianggen

对，今天是个特殊的日子，特殊的日子需要用特殊的方式来纪念，比如,祝大师兄生日快乐！哈哈。
好，不扯皮了，认真写README。

基于mxnet的框架主体，各自独立性良好，为了便于开发和版本管理，我们取名叫BrainMatrix，在我的github隐私repository里。BrainMatrix继承MXNet的C部分，提供具体计算和优化操作。增加了scala kernel，来支持Deep learning，Probabilistic Model，Brainlike Model的计算工作。

Installation for linux

  • 下载BrainMatrix，修改config.mk,主要是USE_CUDA项，如果装了cuda，设为1，同时给出路径；否则设为0；
  • make ， 如果有warning,不管，只要不报错就ok
    mac请参照：mac installation,不过源代码得用Brainmatrix。
  • 如果第二步的make没有因报错而停止，则在目录下会多一个lib文件夹，里面的mxnetlib.so,就是编译好的library
  • 将3中的lib文件夹加入动态链接库，ubuntu的做法是在.bashrc加入：
    export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/home/agen/workspace-c/BrainMatrix/lib
    如果是mac，就是在~/.bash_profile中添加下面一行：
    export DYLD_LIBRARY_PATH="brainmatrix lib 的路径:$DYLD_LIBRARY_PATH"
    是否生效暂时不是那么重要，不过，当报错说找不到libmxnet.so文件的时候，就是这儿设置失败了。
  • 验证是否安装成功
    1 安装maven，在ubuntu下是:sudo apt-get install maven
    2 输入命令：make compile,编译scala内核，需要连接互联网，如果报错，大概是环境问题，建议装jdk1.7.0_121.
  • 如果最后出现BUILD SUCCESS,说明BrainMatrix安装成功。

  • 运行scalakernel：

  • 打开eclipse，import $\rightarrow$ existing maven project $\rightarrow$ 选择scalaKernel $\rightarrow$ 确定

  • 我猜还会有其它小问题，请联系我。如果make成功，那么根目录下example的python文件，有一些是可以run的了

Usage

  • scala程序咋在scalakernel文件夹中，推荐用eclipse scala ide 开发。eclipse提供了代码补全和代码跳转功能，用起来挺顺手的,又因为sxnet是maven项目，eclipse配置maven我比较熟，所以，建议一起用eclipse。
  • 在根目录下新建一个data文件夹，里面存放各种dataSet
  • 下载mnist 文件到data文件夹，然后就可以run sxnet里面的程序了。
  • scalaKernel/src/test文件夹中有一些实例方法，适合上手和学习。
