BrainMatrix Error Set

2016.12.28 LiuXianggen

错误类型

storage.cc cuda is not enable

可能是cuda没装好，
也有可能是编译的时候，没有用cuda编译
也有可能是用cuda编译后，没有make clean，导致cuda没编译通过

Check failed: Device::kDevMask == devmask TBlob.get: device type do not match specified type

自己写的操作报错了，乖乖的换成原装操作，囧！

Check failed: (lhs.ctx()) == (rhs.ctx()) operands context mismatch 2 0 vs. 1

context不一致

symbol 绑定的初始矩阵不能作为输出

[16:35:38] /home/agen/workspace-c/BrainMatrix/dmlc-core/include/dmlc/logging.h:235: [16:35:38] src/symbol/graph_executor.cc:832: Check failed: (info.type) == (kInternalAllocated)
Exception in thread “main” thu.brainmatrix.MXNetError: [16:35:38] src/symbol/graph_executor.cc:832: Check failed: (info.type) == (kInternalAllocated)

java.lang.IllegalArgumentException: requirement failed: Must specify all the arguments in args

Symbol.bind 绑定的(symbol,ndarray)对不一致。

graph_executor.cc:1121: Check failed: (info.ref_count) == (0) Because the last operator is not Loss function, head_gradient is required in calling backward.

最后一层，不是loss层，然后backward函数带的参数不够。

+=操作会改变原始的指针！！！慎用！
