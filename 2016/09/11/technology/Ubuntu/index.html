常用命令

  • ipython notebook –profile-nbserver:开ipython服务器
    在浏览器用地址https://192.168.31.239:port/查看
  • pwd:查看当前目录
  • lshw: 查看电脑硬件详细信息
  • lscpu: 查看cpu
  • htop: 动态监控进程
  • nvidia-smi: 查看gpu
  • find . -name name*: 查找包含name的文件名
  • ssh-keygen: 生成ssh 公钥
  • df -h 查看

jupyter notebook

  1. 远程访问
    此时应该可以直接从本地浏览器直接访问http://address_of_remote:8888就可以看到jupyter的登陆界面。
  2. 建立ssh通道
    如果登陆失败，则有可能是服务器防火墙设置的问题，此时最简单的方法是在本地建立一个ssh通道：
    在本地终端中输入ssh username@address_of_remote -L127.0.0.1:1234:127.0.0.1:8888
    便可以在localhost:1234直接访问远程的jupyter了。

安装

报错 E: Problem with MergeList /var/lib/apt/lists/xxxx

  1. 删除apt-get install 的所有软件状态包,注意提前备份下
    sudo rm /var/lib/apt/lists/* -vfR
  2. 重新加载系统已安装软件新包
    sudo apt-get update
